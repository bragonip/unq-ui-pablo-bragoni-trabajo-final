# TP Final - Piedra, Papel, Tijera, Lagarto, Spock

## Instrucciones para descargar e instalar el proyecto

1. Para poder instalar el proyecto tiene que tener instalados previamente node y npm

2. Descargar el proyecto:

Descargar el zip desde la url o directamente desde la terminal de git clonando el repositorio.

- Si descargó el zip, descomprima el archivo, abra la consola y ubíquese dentro del directorio del mismo.  

- Si eligió clonar el repositorio, ubíquese en la carpeta donde quiera clonarlo y escriba el siguiente comando:

git clone git@github.com:bragonip/unq-ui-pablo-bragoni-trabajo-final.git

3. Instalar las dependencias del proyecto desde la terminal:

$ npm install

Iniciar el proyecto:

$ npm start
